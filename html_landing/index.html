지금바로 시작하기를 누르고 AWS 에서 IAM 키를 발급받아서 AWS Interface 를 시작합니다. 가이드 문서를 참조하여 간단한 사용 방법과 작동 방식에 대해 알아봅니다.


더 알아보기
대시보드 설정

대시보드에서 원하는 서비스 구성을 작성하고 저장합니다. 클릭만으로 원하는 Database 파티션을 만들고 접근 권한을 수정하고. 서버리스 로직을 직접 작성하거나, 회원가입 시 어떤 정보를 입력 받을지 그리고 비밀번호 형식과 인증 방식등을 정할 수 있습니다.

배포 후 모니터링

대시보드에서 설정을 마친후 Overview 화면에서 원하는 언어의 SDK 를 다운받아 클라이언트에 추가합니다. 그 후 얼마나 많은 회원들이 가입했는지 어떤 활동을 하고 있는지 등을 모니터링합니다.

주요기능

클릭 한번에 회원 관리 데이터베이스 관리 미디어 관리
특수로직 관리 및 모니터링 등 모든 핵심 기능이 해결됩니다.

  • 클릭한번에 백엔드가 만들어집니다.
  • 대시보드로 서비스를 관리합니다.
  • 트래픽은 걱정하지 마세요 :)

트래픽 자동 확장

AWSInterface의 Stateless 시스템이 AWS의 고가용성과 트래픽 자동 확장 기능을 최대한 활용할 수 있게 도와줍니다.

완전 병렬 시스템

자동 생성된 API는 AWS Lambda에서 실행되며 함수형으로 작성됩니다. 모든 호출은 병렬로 실행되며 트래픽에 따라 규모가 자동으로 조정 되기 때문에 사용자는 트래픽을 걱정할 필요가 없습니다.

간단한 인터페이스

AWS의 DynamoDB 나 Lambda 와 같은 서비스를 사용하기 위해서는 높은 수준의 배경지식이 필요합니다. 하지만 AWSInterface는 고수준으로 추상화된 인터페이스를 제공하여, AWS 서비스에 대한 별도의 이해 없이도 어플리케이션을 구성할 수 있게 도와줍니다.
