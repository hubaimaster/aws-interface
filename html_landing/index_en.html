Dashboard settings

On the dashboard, create and save the desired service configuration. Just click to create your desired database partition, modify access permissions, create your own serverless logic, specify what information to input when you sign up, and specify the password format and authentication method.

Monitoring

After completing the settings in the dashboard, download the SDK for the desired language from the Overview screen and add it to the client. You can then monitor how many members have joined and what activities they are doing.

Automatically expand traffic

AWSInterface's stateless system helps you make the most of AWS's high availability and traffic auto-expanding capabilities.

Fully parallel system

The auto-generated API runs in AWS Lambda and is written in a functional form. Because all calls are run in parallel and are scaled automatically according to traffic, You do not have to worry about traffic.

Simple interface

To use services like AWS's DynamoDB or Lambda A high level of background knowledge is required. However, AWSInterface provides a high-level, abstracted interface, It helps you configure applications without any understanding of AWS services.
